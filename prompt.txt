ROOT CAUSE of issues 1-3:
The Reconsideration and Appeal modals render inside 
<fieldset disabled> in page.tsx. ALL native form 
controls inside fieldset disabled are blocked — 
inputs, textareas, selects, date pickers.

FIX 1 — Use React Portal for ALL THREE modals
Import ReactDOM:
import ReactDOM from 'react-dom'

Wrap each modal's JSX in a portal so it renders 
in document.body, outside the fieldset disabled:

BEFORE (Reconsideration modal):
{reconsiderationOpen && (
  <div className="fixed inset-0 z-[60] ...">
    ...modal content...
  </div>
)}

AFTER (Reconsideration modal via portal):
{reconsiderationOpen && ReactDOM.createPortal(
  <div className="fixed inset-0 z-[60] ...">
    ...modal content (unchanged)...
  </div>,
  document.body
)}

Apply the same portal pattern to:
- The unlock options modal (unlockOpen)
- The Reconsideration modal (reconsiderationOpen)
- The Appeal modal (appealOpen)

All modal content stays exactly the same.
Only the wrapping changes to createPortal.
