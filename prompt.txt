3) How fmtDate formats dates
- In useReviewInfo.ts, fmtDate returns d.toLocaleDateString("en-US"), e.g., "4/1/2026" (M/D/YYYY). This is not ISO and must be converted to YYYY-MM-DD for type="date".

Implementation

- Added helper near the top of ReviewInfoSection.tsx:
  const toIsoDate = (display: string): string =&gt; {
    if (!display) return ""
    const d = new Date(display)
    if (isNaN(d.getTime())) return ""
    return d.toISOString().split("T")[0]
  }

- Replaced Field with conditional pattern in edit mode using type="date", value converted via toIsoDate, and onChange calling changes.setField("reviewInfo", fieldName, e.target.value) for each field. Non-edit mode continues to render the original Field.
