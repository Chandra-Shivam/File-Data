Applied targeted visual fixes in frontend/src/app/review-queue/page.tsx only, with no backend or functional changes.

BUG 1 — Column headers not visible in review table
- Root cause: The DataTable component supports a theadClassName prop for header styling. It was not being passed, so the header row used the default styles, and after previous edits the header appeared blank.
- Fix: Pass theadClassName to DataTable with the brand header styles (bg-[#1F3864] text-white).

Before:
  <DataTable&lt;ReviewRow&gt;
    columns={columns}
    rows={sortedRows}
    sortBy={sortBy}
    sortDir={sortDir}
    onSort={handleSort}
    selectedId={selectedId}
    getRowId={(r) =&gt; r.id}
    onSelect={setSelectedId}
    onOpen={(r) =&gt; { void handleOpen(r); }}
    tableLabel="Review Queue"
    className="w-full"
    selectedRowClassName="bg-slate-100"
    rowActions={(r) =&gt; (
      ...
    )}
  />

After:
  <DataTable&lt;ReviewRow&gt;
    columns={columns}
    rows={sortedRows}
    sortBy={sortBy}
    sortDir={sortDir}
    onSort={handleSort}
    selectedId={selectedId}
    getRowId={(r) =&gt; r.id}
    onSelect={setSelectedId}
    onOpen={(r) =&gt; { void handleOpen(r); }}
    tableLabel="Review Queue"
    className="w-full"
    theadClassName="bg-[#1F3864] text-white"
    selectedRowClassName="bg-slate-100"
    rowActions={(r) =&gt; (
      ...
    )}
  />

Validation:
- DataTable.tsx definition includes theadClassName in DataTableProps and applies it to the thead element. This is the correct way to style the header.

BUG 2 — Extra empty space below Totals row (Progress Status panel)
- Root cause: The Progress Status card wrapper included a fixed height class (h-full), causing extra vertical space regardless of content height.
- Fix: Remove h-full so the panel sizes to content.

Before:
  &lt;div className="rounded border border-slate-200 overflow-hidden h-full"&gt;

After:
  &lt;div className="rounded border border-slate-200 overflow-hidden"&gt;

No other changes were made. No new packages, and no backend files were touched.
