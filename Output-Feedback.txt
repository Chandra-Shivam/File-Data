Feature: Help Tips Library Maintenance
Route: /maintenance/help-tips
Auth policy: RequireActiveUser

Database table: [CAS_RISKREVIEW_DEV].[dbo].[03_LIBRARY_06_Help Tips]
Columns:
  - Help_tip_id (int, AUTO-INCREMENT PRIMARY KEY — never 
    user-input, DB generates it, read-only everywhere, 
    exclude from Create request DTO)
  - Help_tip_form (string, free text)
  - Help_tip_topic (string, free text)
  - Help_tip (string, stores HTML content)
  - SSMA_TimeStamp (rowversion — exclude from all DTOs)

Special behaviour for Help_tip column:
  - View mode: render as HTML (not raw text)
  - Edit mode: WYSIWYG rich text editor
  - Stored and saved as HTML string in the database
  - User never writes raw HTML manually
