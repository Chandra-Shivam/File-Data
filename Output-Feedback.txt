The dates in the PDF are displaying as ISO timestamps 
(e.g. 2026-04-01T00:00:00). Format all dates in the PDF 
to a readable format (e.g. 4/1/2026).

Add this helper function at the top of ReviewPDF.tsx 
(inside the file, not exported):

const formatDate = (value?: string | null): string => {
  if (!value) return '-'
  const d = new Date(value)
  if (isNaN(d.getTime())) return value
  return d.toLocaleDateString('en-US', {
    month: 'numeric',
    day: 'numeric', 
    year: 'numeric'
  })
}

Apply formatDate() to every date field in the PDF:
- Sample Date
- Start Date
- Completed Date
- Last Annual Review Date
- Next Annual Review Date
- Review Approval Date
