The root cause is confirmed: the RichTextEditor is inside 
a <form onSubmit={handleCreate}> which violates AGENTS.md 
("Never use HTML form tags — use onClick/onChange instead") 
and causes Enter key interference with contentEditable.

Fix in frontend/src/app/maintenance/help-tips/page.tsx only:

1. Remove the <form onSubmit={handleCreate}> wrapper 
   from the Add New section entirely

2. Replace it with a plain <div>

3. Change the Create button from type="submit" to 
   type="button" with onClick={handleCreate}

4. handleCreate logic stays exactly the same — 
   only the trigger changes from form submit to button click

No other changes. No backend files.
